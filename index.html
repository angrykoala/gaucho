Gaucho

Remove Unnecesary Terminals

Download

Gaucho is an open-source, customizable task launcher to run your apps, commands or scripts.

Do you run the same commands every time?
Gaucho allows you to configure your commands once and run them in a non-intrusive interface.
Say goodbye to dozens of terminals.

Check our documentation to get started.

Remove unnecessary terminals

With Gaucho you can organize all your day-to-day tasks, scripts and server to run them quickly, no extra annoying terminals.

Execute commands with one click

Run and stop your commands, apps and scripts from Gaucho with a single click.

Create your tasks

Easily create and configure your tasks.

Features

  • Configure and run your tasks
  • Organize your scripts
  • Export and import tasks
  • Schedule tasks
  • Manage env variables
  • Themes
  • Available for Linux, Windows and Mac
  • Open Source
